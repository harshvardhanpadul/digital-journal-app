# Digital Journal App

A simple journal application built with Node.js, Express, PostgreSQL and EJS.

## Features
- User registration and login
- Password hashing with bcrypt
- Session authentication
- Create journal entries
- Edit and delete entries
- Bootstrap UI

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- EJS
- Bootstrap
- bcrypt
- express-session

## Run Locally

```bash
npm install
nodemon app.js
```
